# flickrinreact
Using the flickr API with React.js

## working version at http://willmayger.uk/flickr
 To use;
  - clone this repository,
  - run npm install.
  - then run `gulp` for developement. (or `npm run startdev`).
  
  ### To deploy:
   - Run `npm run build`
   - Then push to your server and check it out!
